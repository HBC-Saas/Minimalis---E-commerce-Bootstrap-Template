Getting Started.

To save time and to be more productive we use some modern development tools.

Please note: If you aren’t familiar with these tools or you just simple don’t want to use for whatever reasons you can skip the following instruction and keep working with classic HTML / CSS / JS way. You have all files that needed to starts located inside dist folder.

1. Install Node

To start with the project you need to install node in your machine. If you don’t have node installed (you can test by typing node --version command on terminal) you can download from node official site.

2. Install Gulp

After you have installed node on your machine next you need to install gulp cli. If you don’t have gulp cli installed open terminal and type $ npm install –global gulp-cli

3. Install project dependencies

After installing node and gulp you are almost done with installations just one more thing has been left to be installed! open the project with your preferred editor and on root of the project you have package.json file that has all the dependencies of the project. To install project dependencies you need to type $ npm install this command will install all the dependecies from package.json file, and you are done with the installations.

4. Run Gulp

Run $ gulp command on the terminal to start the project. This command is default task which will create and compile the sources to dist folder and will start browsersync on port 3000.

Development

$ gulp task will run following subtasks:

  • copyVendorTask: Copies 3rd party modules to the vendor directory based on our package.json dependencies.
  • sassDevTask: Compiles style.scss into style.css - no sourcemaps, no autoprefixing, no minification.
  • watchTask: Watch the SCSS/JS/HTML for changes, use Live Reload to update browsers instantly, Any changes made to the source files will be compiled as soon as you save the file.

Prodaction

$ gulp dist task run the following subtasks:

  • cleanTask: Clean the dist folder.
  • copyTask: Copies folders from src folder into dist folder.
  • sassBuildTask: Compiles style.scss into style.css - sourcemaps, autoprefixing, minification.
  • minImgTask: Minifies images.
  • cacheBustTask: Generate a string and place it into html link and script tags for cache purpose.
