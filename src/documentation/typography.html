Links

Text color classes apply to all breakpoints, from xs to xl,

  • .text-{value} for xs.
  • .text-{breakpoint}-{value} for sm, md, lg, and xl.

Where value is one of:

  • primary
  • secondary
  • success
  • warning
  • danger
  • info
  • gray
  • light
  • dark

Font Sizes

Font sizes classes apply to all breakpoints, from xs to xl,

  • .font-size-{value} for xs.
  • .font-size-{breakpoint}-{value} for sm, md, lg, and xl.

Where value is one of:

  • display1
  • display2
  • display3
  • display4
  • h1
  • h2
  • h3
  • h4
  • h5
  • h6
