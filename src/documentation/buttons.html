Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Bootstrap documentation

Colors

button color variant classes apply to all breakpoints, from xs to xl,

  • .btn-{value} for xs.
  • .btn-{breakpoint}-{value} for sm, md, lg, and xl.

Where value is one of:

  • primary
  • secondary
  • success
  • warning
  • danger
  • info
  • gray
  • light
  • dark

Example: btn-md-primary

  • Code
